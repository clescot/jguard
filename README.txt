to integrate jGuard(http://www.jguard.net) into your webapp on your application server, you have to look towards
the main documentation reachable at http://jguard.xwiki.com/xwiki/bin/view/Doc/Home
to install quickly jGuard, you can follow instructions of the Quick start guide reachable here:
http://jguard.xwiki.com/xwiki/bin/view/Doc/QuickStartGuide

jGuardExample use 3rd libraries which endorse their own license.
all licenses are reachable in the 3rdLibrariesLicenses directory.
jGuardExample use:
- Struts(http://struts.apache.org)
- SiteMesh(http://www.opensymphony.com/sitemesh/)
- Skidoo css layout (http://webhost.bridgew.edu/etribou/layouts/index.html)
- 'crystal clear' icon set (http://www.everaldo.com/crystal.html)
- 'nuvola' incon set (http://www.icon-king.com/) 
- 'ie7' javascript library(http://dean.edwards.name/IE7/)
- bouncyCastle security library(http://www.bouncycastle.org/)
- dom4j(http://www.dom4j.org/)
- junit(http://junit.org/)
- JCaptcha(http://jcaptcha.net)
